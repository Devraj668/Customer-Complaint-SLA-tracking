This project demonstrates a banking complaint tracking system to monitor SLA compliance, issue categories, and agent performance using Excel dashboards and MySQL queries.

## Project Objective
To design a simple banking complaint tracking system that monitors SLA compliance
and improves operational visibility.

## Challenges Faced
- Handling NULL values during CSV import
- SLA calculation for open vs closed tickets
- Designing meaningful KPIs for operations

## Solution
- Standardized NULL handling
- SQL-based SLA evaluation
- Excel dashboard for visualization

## Future Improvements
- Automated SLA alerts
- Integration with ticketing tools
- Predictive complaint analysis
