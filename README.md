# Customer Complaint & SLA Tracking System

## Overview
This project simulates a banking customer service environment to track
complaints, monitor SLA compliance, and analyze service performance.

## Tools Used
- Microsoft Excel
- MySQL
- SQL
- Pivot Tables

## Key KPIs
- Total Tickets
- SLA Compliance %
- Complaint Priority %
- Issue-wise Complaints Count
- Agent-wise Ticket Handling Data

## Dashboard
An Excel dashboard was created to visualize KPIs and SLA breaches.
